Google Sign-In Membutuhkan Server

Keamanan Google OAuth tidak mengizinkan login melalui protokol berkas langsung (file://). Anda perlu menjalankan aplikasi di server lokal (http://localhost).

1

Jalankan berkas jalankan-aplikasi.bat di folder utama aplikasi.

2

Aplikasi otomatis terbuka di browser pada alamat http://localhost:3000.

3

Klik kembali tombol Masuk dengan Google.

Atau masuk menggunakan Email & Password

Domain Belum Diotorisasi

Domain domain.com belum diotorisasi di Firebase Console untuk proyek reuniakbar.

1

Buka Firebase Console dan pilih proyek reuniakbar.

2

Pilih menu Authentication di samping kiri, lalu klik tab Settings.

3

Scroll ke bagian Authorized Domains, lalu klik tombol Add Domain.

4

Masukkan domain: domain.com lalu klik Save.
